## 📖 توضیحات

یک **مدیر دیتابیس MySQL حرفه‌ای** با پشتیبانی دوسویه (انگلیسی و فارسی). این ابزار کمک می‌کند تا به راحتی داده‌های دیتابیس خود را به فایل‌ها صادر کنید و بازگرد انتقال دهید.

### ✨ ویژگی‌های اصلی

- ✅ **رابط دوسویه** - پشتیبانی کامل انگلیسی و فارسی
- ✅ **ورودی دوست‌داشتنی** - پرامپت‌های تمیز و مرتب با مقادیر پیشفرض
- ✅ **صادرات هوشمند** - تجزیه و ذخیره‌سازی تمام جداول دیتابیس
- ✅ **درج ایمن** - بازگردانی داده‌ها با مدیریت خطاهای مناسب
- ✅ **امنیت** - محافظت از حملات SQL Injection
- ✅ **رابط حرفه‌ای** - فرمت‌بندی زیبا کنسول
- ✅ **مدیریت خطاهای کامل** - مدیریت جامع استثناها
- ✅ **طراحی شی‌گرا** - ساختار کد تمیز و قابل نگهداری

## ⚠️ نکات مهم

### 1. ابتدا پشتیبان بگیرید
```bash
# همیشه قبل از درج داده‌ها، پشتیبان دیتابیس را ذخیره کنید
mysqldump -u root -p database_name > backup.sql
```

### 2. خطر داده‌های تکراری
```
⚠️  هشدار: درج داده‌ها ممکن است داده‌های تکراری ایجاد کند
    
راه‌حل:
1. ابتدا جداول را خالی کنید (TRUNCATE TABLE)
2. یا از REPLACE INTO به جای INSERT استفاده کنید
3. برای تضاد کلید اولیه بررسی کنید
```

### 3. محدودیت‌های کلید خارجی
```
⚠️  اگر دیتابیس شما کلیدهای خارجی دارد:

1. بررسی کلید خارجی را در طول درج غیرفعال کنید
2. ابتدا جداول parent را درج کنید
3. سپس جداول child را درج کنید
```

### 4. دیتابیس‌های بزرگ
```
⚠️  برای دیتابیس‌های بسیار بزرگ:

در نظر بگیرید:
- استفاده از صفحه‌بندی
- تقسیم درج به دسته‌های کوچک
- استفاده از دامپ‌های SQL مستقیم (mysqldump)
```

## 🐛 حل مشکلات

### خطا: `Access denied for user 'root'@'localhost'`

**مشکل:** رمز عبور یا کاربر اشتباه

**راه‌حل:**
```bash
# بررسی مشخصات MySQL
mysql -u root -p -h localhost

# تأیید وجود دیتابیس
SHOW DATABASES;
```

### خطا: `Unknown database 'my_database'`

**مشکل:** دیتابیس وجود ندارد

**راه‌حل:**
```bash
# ایجاد دیتابیس
mysql -u root -p
CREATE DATABASE my_database;
```

### خطا: `No module named 'mysql'`

**مشکل:** mysql-connector-python نصب نشده است

**راه‌حل:**
```bash
pip install mysql-connector-python
```

### خطا: `Pickle file not found`

**مشکل:** داده‌های دیتابیس هنوز صادر نشده‌اند

**راه‌حل:**
```bash
# ابتدا گزینه 1 را اجرا کنید (صادرات)
# سپس گزینه 2 را اجرا کنید (درج)
```

### خطا: `Duplicate entry`

**مشکل:** داده‌ها در دیتابیس وجود دارند

**راه‌حل:**
```bash
# جداول را قبل از درج خالی کنید
TRUNCATE TABLE table_name;

# یا داده‌های خاص را حذف کنید
DELETE FROM table_name WHERE id > 0;
```
